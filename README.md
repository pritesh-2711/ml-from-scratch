## Machine Learning - from maths to machines

This repository will be used to explain the following concepts with mathematical solutions and coding.
Codes will be in jupyter notebooks as our focus here is not going to be application design; but building models from scratch. And that can be a messy notebook at times.

----

### Topics to be covered :

The learning path will be done via 4 parts

#### Part A : 
- Probability
- Statistics

#### Part B :
- Linear Algebra
- Matrices
- Vectors

#### Part C :
- Regression
- Timeseries Forecasting

#### Part D :
- Binary Classification
- Multiclass Classification

#### Part E :
- Clustering
- Segmentation / Profiling Analytics

#### part F :
- Recommendation Engines

----
